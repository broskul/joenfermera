# Jocelyn Ortega Enfermera

Sitio estático profesional para Jocelyn Ortega, Enfermera Universitaria.

## Archivos principales
- `index.html`: estructura y contenido del sitio.
- `styles.css`: diseño responsive.
- `script.js`: menú móvil y año del footer.
- `assets/images/`: fotografías optimizadas.
- `blog/`: artículos SEO sobre heridas, estética ética, pabellón y PolyMem.
- `robots.txt` y `sitemap.xml`: archivos para indexación en buscadores.
- `SEO-Google-Bing.md`: guía para registrar el sitio en Google Search Console, Google Business Profile, Bing Webmaster Tools y Bing Places.
- `ContextIA/Instrucciones.md`: contexto fuente del proyecto, sin mostrarse en la UI.

## Vista local
Abrir `index.html` directamente en el navegador.
